== Description ==

This plugin removes that pesky site icon. That is it. Simplicity incarnate.

== Installation ==

Follow these steps to install our Wordpress Plugin via the \"Add New Plugin\" method:

1. Login to your Wordpress admin dashboard: The URL typically looks like this: http://yourdomain.com/wp-admin

2. Navigate to the Plugins menu: In the left-hand side admin panel, look for the \'Plugins\' menu item and click on it.

3. Add New Plugin: On the \'Plugins\' page, you will find an \'Add New\' button at the top. Click on it.

4. Upload the Plugin: In the \'Add Plugins\' page, click on the \'Upload Plugin\' button at the top. This will allow you to upload your plugin\'s .zip file.

5. Select the Plugin\'s zip file: Click on the \'Choose File\' button, navigate to the location of the .zip file of our plugin on your computer, select it, and click \'Open\'. After the file is selected, click on \'Install Now\' button.

6. Install the Plugin: Wordpress will now upload and extract the plugin .zip file for you. After this, you will see a \'Plugin installed successfully\' message.

7. Activate the Plugin: After the plugin is installed successfully, you need to activate it. Click on the \'Activate Plugin\' link to activate the plugin on your Wordpress site.
